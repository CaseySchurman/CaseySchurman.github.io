UI Design & Development

I'm passionate about UI Design and Development because a well designed and implemented user interface can make all the difference in making a software application successful. If it's bad, the software application will likely fail.

 

Software QA & Testing

I love QA and Testing in Software because they're similar to Ringo Starr in The Beatles. It doesn't get all of the attention and spotlight, but without it the software development process wouldn't become one of the best bands of all time (whoops, I mean wouldn't be an effective software development process).

 

Mobile App Development

I love that with mobile application development I can have an idea for something I'd like to have on my phone, and if it isn't there, I can just create it. The problem will be solved, and one less thing to stress or worry about is always nice.

Cooking With Engineers

Developed with team of 4 other students. Website is a social network where users could store, share, and rate cooking recipes. We used the Rails Framework which includes Ruby, Javascript, HTML, SCSS and PostgreSQL. During this time we were focusing heavily on the Agile development method over the Waterfall method. Most of the pictures (including for the background) are missing from the heroku site since they were done through a trial version of Amazon web services, as well as the database being out of date and not updated. But it gives an idea for what the site is.
